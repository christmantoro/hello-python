# hello-python

Curated index of Python tools for Meta Ads (Facebook & Instagram) marketing — ecosystem directory pointing to community repos and in-house projects.

## Purpose

Acts as starred landing page so marketers and developers discover Python tools for Meta Ads workflows. No runnable code lives here — links only.

## Structure

Two repo types listed in README:
- **Our Projects** — repos built and maintained by this team, linked when published
- **Community repos** — third-party Python tools organized by category

## Categories

Eight workflow categories cover end-to-end Meta Ads use cases: API connectors, analytics/reporting, lead ads, data pipelines, budget optimization, attribution, automation, and learning resources.

## Contributing

External contributors submit repos via PR against README.md. Criteria: Python primary language, active maintenance, solves a Meta Ads problem. Submission format in CONTRIBUTING.md.

## Roadmap

Our Projects section will expand as individual repos are built:
- `meta-ads-insights` — Marketing API data extraction
- `meta-lead-pipeline` — Lead Ads → CRM pipeline  
- `meta-ads-reporter` — Automated reporting
- `meta-audience-builder` — Audience management
