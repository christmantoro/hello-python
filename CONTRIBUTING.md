# Contributing

## How to add a repo

1. Fork this repo
2. Add entry to the correct category table in `README.md`
3. Open a PR with title: `Add: [repo-name]`

## Entry format

```markdown
| [repo-name](https://github.com/owner/repo) | One-line description of what it does | ![Stars](https://img.shields.io/github/stars/owner/repo?style=flat-square) |
```

## Requirements

- Python must be primary language
- Repo must solve a Meta Ads / Facebook Ads / Instagram Ads workflow problem
- Last commit within 12 months (or historically significant/widely used)
- Public repo with a README

## New category?

If no existing category fits, propose one in your PR description. Keep category names short and action-oriented.
